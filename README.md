# Velocity CLI

The official CLI for the [Velocity](https://github.com/velocitykode/velocity) Go web framework.

## Installation

**Requirements**: Go 1.25 or higher

```bash
go install github.com/velocitykode/velocity-cli@latest
```

## Usage

Create a new project:

```bash
velocity new myapp
cd myapp
go run main.go
```
